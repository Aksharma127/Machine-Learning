# Machine Learning Portfolio

Curated machine learning notebooks organized for review, reuse, and portfolio presentation.

## Structure

```text
Machine-Learning-main/
├── assignments/   # Coursework and graded notebooks
├── labs/          # Guided lab sessions
├── notebooks/     # Core algorithm and EDA tutorials
├── projects/      # End-to-end project notebooks
├── practice/      # Short practice notebooks
├── experiments/   # Exploratory and experimental work
└── archive/legacy/ # Duplicate, broken, or superseded notebooks
```

## Highlights

- Clean snake_case notebook names for easier browsing.
- Topic-based folders so recruiters can find projects quickly.
- Legacy copies and broken notebooks moved out of the main view.
- Retained datasets are kept at the repository root for convenience.

## Notes

- `projects/` contains the most polished work, including survival prediction, fraud detection, geospatial modeling, diffusion modeling, and transfer learning.
- `notebooks/` contains algorithm walkthroughs such as logistic regression, decision trees, random forests, SVM, Naive Bayes, and regression techniques.
- `archive/legacy/` contains duplicate, incomplete, or low-signal notebooks that are preserved only for reference.

## Dataset

- `fake_banking_app_dataset.csv` is the primary CSV asset currently included in the repository.
